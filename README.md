# 💓 Heart Disease Prediction App

This ML app predicts heart disease based on user inputs using Logistic Regression, Random Forest, and KNN. It also includes Streamlit visualization.

## 🔍 Features
- Age, cholesterol, chest pain, heart rate, etc.
- ML Models: Logistic Regression, Random Forest, KNN
- Visualization using Matplotlib and Seaborn

## 🚀 Run the App
```bash
streamlit run app.py
